# Exercise2-6
Digite um número de 0 a 100

O rograma lerá um valor qualquer e apresentará uma mensagem dizendo em qual dos seguintes intervalos ([0,25], (25,50], (50,75], (75,100]) este valor se encontra. Obviamente se o valor não estiver em nenhum destes intervalos, será impressa a mensagem “Fora de intervalo”
